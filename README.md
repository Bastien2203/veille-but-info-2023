## Getting Started

Generate migrations and data :
```bash
yarn prisma:migrate
```

```bash
yarn generate:data
```

Install dependancies

```
yarn install
```

Run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
